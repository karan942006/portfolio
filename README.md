## 🧑‍💻 About This Repository

This repository hosts the source code for my **personal portfolio website**, built to showcase my projects, skills, and journey as an aspiring Android Developer. It's a fully responsive, single-page site built with vanilla **HTML, CSS, and JavaScript** — no frameworks, just clean handcrafted code.

🔗 **Live Site:** [karan942006.github.io/portfolio](https://karan942006.github.io/portfolio/)

---

## ✨ Features

- 🎨 Custom-designed UI with a refined editorial aesthetic (Playfair Display + DM Sans + JetBrains Mono)
- 📱 Fully responsive across mobile, tablet, and desktop
- ⚡ Smooth scroll-reveal animations and a live scroll progress bar
- ⌨️ Animated typing effect in the hero section
- 🗂️ Sections for About, Skills, Projects, Education, GitHub Activity, and Certificates
- 📨 Working contact form (powered by FormSubmit) for internship/job inquiries
